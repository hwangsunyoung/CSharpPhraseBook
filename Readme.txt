이 솔루션에 포함된 코드에 관하여

1. 이 책 게재한 예제 코드는 최대한 실행되도록 작성했지만 일부 코드는 실행되지 않을 수도 있습니다.
    이 부분은 양해하기 바랍니다.

2. 원칙적으로 책에 있는 하나의 절에서 하나의 프로젝트를 진행하도록 했지만 몇몇 장에서는
    하나의 장에서 하나의 프로젝트를 진행하는 곳도 있습니다.

3. 각각의 코드에는 코드 번호를 주석이나 속성을 통해 기재했습니다.

4. 코드 중에 일부는 편의 상 주석 처리한 것도 있습니다.

5. 많은 콘솔 응용 프로그램 형식의 프로젝트는 공통적으로 SampleCodeRunner.cs를
    이용합니다. 이 코드가 각각의 클래스에 정의된 메서드(public void)를
    호출해서 메서드를 연속으로 호출합니다.

6. 일부 코드는 서적과 다른 것이 있습니다. 양해하기 바랍니다. 본질적인 부분에는
    차이가 없습니다.

7. 책에 게재한 여러 개의 코드를 하나의 클래스로 정리했기 때문에 고정값 문자열이나
    배열을 초기화하는 코드에는 중복되는 부분이 있지만 책에 게재한 내용에 최대한 맞추기
    위해 일부러 중복시킨 것입니다.
    실제로 개발할 때는 중복이 없도록 고려하기 바랍니다.

8. 18장 19장에 있는 예제 코드는 이 솔루션에 포함돼 있지 않습니다. 양해하기 바랍니다.

9. 예제 코드에는 UWP 응용 프로그램의 프로젝트가 존재합니다. UWP 응용 프로그램을
    실행하려면 Windows10이 설치돼 있어야 합니다. 그리고 Visual Studio에 '유니버셜 Windows 개발 도구'가
    설치돼 있어야 합니다.

     [로컬 컴퓨터]로 지정해서 디버깅하기 기바랍니다.
